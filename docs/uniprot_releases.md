# UniProt Release History

Documentation of UniProt/Swiss-Prot releases used in ToxProt25.

## Background

The UniProt Consortium formed in **2002** (SIB, EBI, PIR), but the first release—**UniProt 1.0**—came on **December 15, 2003**. XML support was added in Release 2.0 (July 2004).

### Version Numbering

- **2003–2009**: Numeric versions (1.0 through 15.15)
- **2010+**: Calendar-based `YYYY_MM` format

## Why 2005–2025?

- **2003–2004**: XML not available or incomplete; ToxProt annotation nascent
- **2005+**: Full XML support, meaningful toxin coverage

## FTP Archive

Historical releases: https://ftp.uniprot.org/pub/databases/uniprot/previous_major_releases/

ToxProt25 downloads `uniprot_sprot.xml.gz` from each release's `knowledgebase/complete/` directory.
